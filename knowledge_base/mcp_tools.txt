MCP 工具使用说明

1. mcp-server-fetch 工具
   - 功能：获取公开网页内容
   - 使用方法：提供 URL 参数
   - 限制：仅访问单页，遵守网站条款
   - 返回：网页的文本内容

2. @modelcontextprotocol/server-filesystem 工具
   - 功能：文件系统操作
   - 支持操作：读取、写入、列出文件
   - 安全限制：只能在允许的目录内操作
   - 配置：通过 allowDir 参数指定允许的目录

3. 工具调用流程
   - LLM 分析用户需求
   - 选择合适的工具
   - 构造工具调用请求
   - 执行工具并获取结果
   - 将结果返回给 LLM 继续处理

4. 错误处理
   - 网络错误：重试或报告错误
   - 文件错误：检查权限和路径
   - 工具错误：记录日志并继续
